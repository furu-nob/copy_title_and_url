# copy_title_and_url

## Chrome Plugin 
Plugin to copy page title and URL in Wiki format.

* Option Example
  * Markdown <-- default
  ```（[{title}]({url})）```
  * Textile  
  ```（"{title}":{url}）```
  * Custum (You can Write lile bellow.)  
   {title}, {url}

* やりたいこと
  * notifications にコピーした内容を追加
  * エラーハンドリング → http https 以外のページではエラーを出したい
